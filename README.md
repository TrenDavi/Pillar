## Emu-Arch
RISC-V Architecture Emulator in Verilog


### Memory Map

| Range                     | Description                                         |
| ------------------------- | --------------------------------------------------- |
| 0x0000_0000 - 0x0000_0000 | Reset Vector                                        |
| 0x0000_0001 - 0x0000_FFFF | 64KB RAM                                            |
